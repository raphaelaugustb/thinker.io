## Run App api

Clone repo

```bash
  git clone https://github.com/raphaelaugustb/thinker.io/
```

Go on directory

```bash
  cd thinker.io/backend/thinker.io
```

Install Dependencies

```bash
  mvn clean install package 
```

Init app

```bash
  mvn spring-boot:run 
```
